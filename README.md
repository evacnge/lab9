Lab Summary

In this lab, we worked on a loan dataset using Decision Tree and Random Forest classification models. First, we imported the required libraries and loaded the dataset. Then, we explored the data using info(), head(), and describe() to understand the dataset structure.

We performed Exploratory Data Analysis (EDA) by creating histograms, countplots, and relationship plots to understand patterns between features such as FICO score, interest rate, credit policy, and loan payment status.

After that, we prepared the data by converting categorical features into dummy variables and split the dataset into training and testing sets.

Then, we trained a Decision Tree model and evaluated its performance using classification reports and confusion matrices.

Finally, we trained a Random Forest model and compared its performance with the Decision Tree model. The Random Forest performed better overall.
